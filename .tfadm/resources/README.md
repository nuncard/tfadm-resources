# .root

## Synopsis

```
tfadm COMMAND [OPTIONS] .root [{environment}]...
```

## Description

Root configuration, shared across all resources.

## Properties

- **`environment`**

  Environment name.

- **`domain`**

  Root domain name used to identify services provided through the Internet.
