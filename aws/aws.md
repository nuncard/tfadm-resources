# .aws

Amazon Web Services (AWS)

## Usage

```
tfadm COMMAND [OPTIONS] .aws [{environment}]...
tfadm COMMAND [OPTIONS] .aws [{domain}]...
```

## Properties

- **`profile`**

  Use a specific profile from your credential file.

- **`region`**

  The name of the AWS region.
